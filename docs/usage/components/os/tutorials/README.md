# Hands-On Tutorials

Welcome to our instructional tutorials for openUC2 OS! Here, we'll learn about how to use openUC2 OS — by using it!

Our practical tutorials are organized by a progression of the things you need to learn to do in order to use openUC2 OS effectively.

:::tip

If you're here because you want to learn about what openUC2 is, our [in-depth explanatory documentation](../explanations/README.md) may be more useful for you.

:::
